# Who's my person?
- IT Students
- Software Developers
- Technical Self-Learners

---

# Their problem
- Fragmented daily logs and lecture notes across different apps.
- Lack of proper Markdown and syntax highlighting.
- Inefficient and slow information retrieval.

---

# What I built
- **DevPulse**: A Web-Based Knowledge Management System (KMS).
- Features a Markdown Workspace and a Dual-Hub Repository.
- Acts as both a private daily log and a public portfolio platform.

---

# How I built it
- **Tech Stack**: Next.js 14 (App Router Monolith), Tailwind CSS, and Supabase (PostgreSQL + Auth).
- **MCP**: Filesystem configured for direct local read/write access.
- **Skill**: `laravel-react` (Best practices for clean API and UI).
- **Agent**: `devpulse-builder` (Expert Full-Stack developer assistant).

---

# Why it matters
- Unifies scattered knowledge into a single, high-performance platform.
- Transforms raw daily engineering notes into a demonstrable, professional Full-Stack portfolio.
